自动给方法添加注释： 输入 /**，然后回车
导入包，自动修正： ALT + ENTER
查找类： CTRL + N

格式化代码： CTRL + ALT + L

快速生成代码（如get/set方法，构造方法等）： ALT + insert
单行注释： CTRL + / （//）
多行注释： CTRL + SHIFT + / （/* */）
撤销到上一步： CTRL + Z
撤销到下一步： CTRL + SHIFT + Z
替换文本： CTRL + R
查找当前页文本： CTRL + F
查找全局文本： CTRL + SHIFT + F
方法参数提示： CTRL + P
最近打开的文本： CTRL + E

跳到大括号的开头/结尾： CTRL + [ 或 ]
